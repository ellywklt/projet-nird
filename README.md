# Enigma_NDI_2025


## Ouverture du site

Pour lancer le site, allez dans le dosier `flask_app`, ouvrez un terminal, et tapez la commande `flask run`.

Lancer le site avec le lien `http://127.0.0.1:5000`

## Utilisation

Les flèches `droite` et `gauche` du clavier, ainsi que les boutons aux bords de l'écran, permettent de faire tourner le carrousel.
